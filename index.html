🧩 Chat Widget Integration Guide

Berikut adalah cara mengintegrasikan Chat Widget ke dalam website Anda.

📦 1. Tambahkan Script

<script src="https://yourdomain.com/chat-widget.js"></script>

⚙️ 2. Siapkan Tempat untuk Widget

<div id="my-widget"></div>

🚀 3. Inisialisasi Widget

<script>
  window.renderChatWidget("my-widget", {
    token: "xxx",
    user_profile: {
      name: "Fani Adi Frihandoko",
      nip: "012",
      gender: "Laki-laki",
      birthdate: "01/01/1990"
    },
    cakra_chat_api_key: "ba166318-3de1-4e48-9d52-7f3075acb142"
  });
</script>

📘 Props yang Didukung

  • token (string) – Token autentikasi user
  • user_profile (object) – Profil user
  • cakra_chat_api_key (string) – API key dari sistem Cakra

🎉 Selesai! Chatbot Anda kini aktif di halaman Anda.
