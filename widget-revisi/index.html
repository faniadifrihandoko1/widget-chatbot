üìã Panduan Instalasi

Langkah 1: Persiapan

1 Download file widget: Unduh file altius-chat-widget.min.js dan upload ke server Anda
2 Siapkan kredensial API: Pastikan Anda memiliki chat_api_key dan chat_api_tenant yang valid

Langkah 2: Integrasi ke Website

Tambahkan 2 script berikut ke halaman HTML Anda, sebaiknya sebelum tag </body>:

<!-- Script konfigurasi --> <script> window.session_id = ""; // Optional - kosongkan untuk auto-generate window.chat_api_key = "your_api_key_here"; // Required - API key Anda window.chat_api_tenant = "your_tenant_id_here"; // Required - Tenant ID Anda window.personal_data = "Data Personal User"; // Required - Data personal user </script> <!-- Widget script --> <script src="altius-chat-widget.min.js"></script>
‚úÖ Selesai! Widget akan otomatis muncul di pojok kanan bawah halaman Anda.

üîß Troubleshooting

Masalah Umum

1 Widget tidak muncul:
  • Pastikan file altius-chat-widget.min.js dapat diakses
  • Cek console browser untuk error JavaScript
  • Pastikan semua variabel required sudah diset
2 Pesan tidak terkirim:
  • Cek koneksi internet
  • Pastikan API key valid dan tidak expired
  • Cek network tab di developer tools untuk error API
3 Session tidak terbuat:
  • Pastikan endpoint create_session dapat diakses
  • Cek header qubisa-token-key sudah benar
  • Pastikan tidak ada CORS issue

Debug Mode

Untuk debugging, buka Developer Tools (F12) dan cek tab Console untuk log error.
